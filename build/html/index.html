Información de las Rutas de la Backend

Descripción de las columnas:

  • Ruta: representa la ruta de la API
  • Método: indica el método HTTP utilizado
  • Descripción: proporciona una breve explicación de la funcionalidad
  • Body: muestra un ejemplo del "body" de la solicitud en formato JSON

Usuarios (user)

Ruta Método Descripción Body
/user GET Obtiene todos los usuarios -
/user/:id GET Obtiene un usuario específico por ID -
/user POST Crea un nuevo usuario
{ "name":"ejemplo2", "email":"ejemplo2@ejemplo.ucn.cl" }
/user/:id PUT Actualiza un usuario existente por ID
{ "name":"ejemplo2", "email":"ejemplo2@ejemplo.ucn.cl" }
/user/:id DELETE Elimina un usuario específico por ID -

Estanques (pond)

Ruta Método Descripción Body
/pond GET Obtiene todos los estanques -
/pond/:id GET Obtiene un estanque específico por ID -
/pond POST Crea un nuevo estanque
{ "id_equipment": 1, "id_microalgae_state": 1, "microalgae_name": "prueba", "min_NTU": 0.1, "max_NTU": 0.2 }
/pond/:id PUT Actualiza un estanque existente por ID
{ "id_equipment": 1, "id_microalgae_state": 1, "microalgae_name": "prueba", "inoculation_date": "2023-01-02", "harvest_date": "2023-01-04", "min_NTU": 0.1, "max_NTU": 0.2 }
/pond/:id DELETE Elimina un estanque existente por ID -
/pond/state/:id GET Obtiene el estado actual del estanque -

Reportes (report)

Ruta Método Descripción Body
/report GET Obtiene todos los reportes -
/report/:id GET Obtiene los reportes específico por de un estanque, la ID solicitado es del estanque -
/report POST Crea un nuevo reporte
{ "pond_id": 1, "illumination": 100.5, "NTU": 4.2 }
/report/:id PUT Actualiza un reporte existente por ID
{ "pond_id": 1, "date": "2023-01-05", "time": "09:35:00", "illumination": 100.5, "NTU": 4.2 }

Equipo (equipment)

Ruta Método Descripción Body
/equipment GET Obtiene todos los equipos -
/equipment/:id GET Obtiene un equipo en particular, la ID solicitado es del estanque. -
/equipment POST Crea un nuevo equipo
{ "start_time": "10:15:00", "end_time": "15:45:00" }
/equipment/:id PUT Actualiza un el tiempo de inicio y termino de un equipo existente por ID
{ "start_time": "10:15:00", "end_time": "15:45:00" }
/equipment/:id DELETE Elimina un equipo por id -
/equipment/openbomb/:id PUT Enciende la bomba de agua del equipo por id -
/equipment/closebomb/:id PUT Apaga la bomba de agua del equipo por id -
